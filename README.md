Project Title

Super Store Sales Analytics Dashboard

📝 Short Description

An interactive data visualization dashboard that analyzes retail sales data across regions, categories, and time periods. It provides insights into sales performance, profit trends, shipping modes, and customer segments using dynamic charts and KPIs.
Purpose of the Project
To track and monitor overall sales, profit, and orders
To identify high-performing regions and categories
To analyze monthly trends (Year-over-Year)
To understand customer behavior (segment & payment mode)
To support data-driven business decisions
🛠️ Tech Stack
Frontend / Visualization Tools
Power BI / Tableau (most likely used)
Programming (optional)
Python (Pandas, Matplotlib, Seaborn)
Database
MySQL / MongoDB / Excel
Other Tools
